# GitHub User Finder

## Автор
Ткаченко Иван
## Описание
Графическое приложение на Python (Tkinter) для поиска пользователей GitHub по никнейму. 
Приложение позволяет искать пользователей через официальный GitHub API, просматривать краткую информацию и сохранять понравившихся пользователей в список "Избранное". 
Данные избранного сохраняются локально в формате JSON.

## Функциональность
1. **Поиск**: Ввод никнейма и получение данных (логин, имя) через API.
2. **Валидация**: Проверка на пустое поле ввода.
3. **Избранное**: Добавление пользователей в список, сохранение в `favorites.json`, загрузка при запуске.
4. **Интерфейс**: Простое GUI окно с полями ввода и списками.

## Требования
- Python 3.x
- Стандартные библиотеки: `tkinter`, `json`, `urllib`

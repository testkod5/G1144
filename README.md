## 📝 Описание программы

**Random Task Generator** - это десктопное приложение на Python с графическим интерфейсом, которое помогает генерировать случайные задачи из различных категорий (учёба, спорт, работа, дом, саморазвитие).

Приложение позволяет:
- 🎲 Генерировать случайные задачи
- ➕ Добавлять свои собственные задачи
- 📜 Просматривать историю всех сгенерированных задач
- 🔍 Фильтровать историю по категориям
- 💾 Сохранять историю в JSON файл
- 🗑️ Удалять отдельные задачи или всю историю

## 🚀 Функциональные возможности

### 1. Генерация задач
- Случайный выбор задачи из выбранной категории
- Возможность выбора "Все категории" для полной случайности
- Визуальное отображение сгенерированной задачи

### 2. Управление задачами
- **50+ предопределенных задач** в 5 категориях
- Добавление пользовательских задач с выбором категории
- Проверка на пустую строку при добавлении

### 3. История
- Автоматическое сохранение всех сгенерированных задач
- Отображение времени генерации
- Подсветка пользовательских задач (желтым цветом)

### 4. Фильтрация
- По категориям (Учёба, Спорт, Работа, Дом, Саморазвитие)
- Отдельный фильтр для пользовательских задач

### 5. Статистика
- Общее количество задач
- Количество добавленных пользовательских задач

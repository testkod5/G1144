# Random Quote Generator

## Автор
**Ткаченко Иван**

## Описание
Приложение для генерации случайных цитат с фильтрацией по автору и теме, сохранением истории в JSON.

## Функции
- 🎲 Генерация случайных цитат
- 📚 11 предустановленных цитат
- ➕ Добавление своих цитат
- 🔍 Фильтрация по автору и теме
- 📜 История всех сгенерированных цитат
- 💾 Автосохранение в JSON
- 🗑️ Удаление цитат из истории

## Установка и запуск

```bash
# Клонирование
git clone https://github.com/ВАШ_ЛОГИН/random-quote-generator.git
cd random-quote-generator

# Запуск
python quote_generator.py

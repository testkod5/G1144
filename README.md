## Описание программы
Expense Tracker - это десктопное приложение для отслеживания личных расходов с графическим интерфейсом.

### Основные возможности:
- ✅ Добавление расходов (сумма, категория, дата)
- ✅ Фильтрация по категории и периоду дат
- ✅ Подсчет общей суммы расходов
- ✅ Удаление записей
- ✅ Сохранение данных в JSON
